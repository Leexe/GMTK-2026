# Game Manager v1.1

## v1.1
- Added `QuestManager` integration.
- Added event triggers for `OnInteractableEnter`, `OnInteractableExit`, and `OnItemPickUp`.
- Added time scale management methods (`TimeSlow`, `CancelTimeSlow`, `SetPaused`, `SetBaseTimeScale`, `SetSimulationTimeScale`).
- Refactored `LockCursor` and `UnlockCursor` into static methods.
